Step 1: Installer micro-ROS

Installation des pr√©requis:

sudo apt install software-properties-common
sudo add-apt-repository universe
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
sudo apt update
mkdir -p ~/micro_ros_ws/src
cd ~/micro_ros_ws
git clone -b foxy https://github.com/micro-ROS/micro_ros_setup.git src/micro_ros_setup

Step 2: Installation des dependencies

ROS2 dependencies

sudo apt install python3-colcon-common-extensions python3-vcstool

Rosdep

rosdep init
rosdep update
rosdep install --from-paths src --ignore-src -y

Step 3: Setup micro-ROS

Build ROS2

colcon build
source install/local_setup.bash

Step 4: Installation firmware & Agent

Firmware

ros2 run micro_ros_setup create_firmware_ws.sh host
source install/local_setup.bash

Agent

ros2 run micro_ros_setup create_agent_ws.sh
ros2 run micro_ros_setup build_agent.sh
source install/local_setup.bash

Step 5: Run

Default baud rate is 115200

ros2 run micro_ros_agent micro_ros_agent serial --dev /dev/ttyACM0

In case of error :

Check if ACM0 is busy

lsof | grep ttyACM0

Kill corresponding process

kill "process number"

Step 1: Complex Function

Let's create a more advanced function:

function processData(data) {
    return data.map(item => item * 2)
               .filter(item => item > 10);
}

const result = processData([2, 4, 6, 8, 10]);
